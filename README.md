# SHLib 上图助手

## 读书的那些事儿

书非借不能读也，此话一点不假。

以前在网上倒是买了不少书，可真正读完的却屈指可数，大部分都是放在书架上吃灰。偶尔想起来翻开一本读几页，而后却没有坚持读完，着实惭愧。

而借的书却不同，还书日期自然地成为了deadline，它会督促着你每天花点时间把书看完。几年前在上海图书馆办了张借书证，这些年下来断断续续竟然也读完了不少的书，效果着实可观。

后来索性在上图办了张参考外借的借书证，可选择的书多了很多，但......

再后来便有了这个`上图助手`。
